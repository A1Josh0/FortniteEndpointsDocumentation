## Events Service - Event Window Leaderboard

URL: https://events-public-service-live.ol.epicgames.com/api/v1/leaderboards/:GameId/:EventId/:EventWindowId/:AccountId?rank=1&showLiveSessions=true \
Method: GET \
Auth Required: Yes

### Parameter

`GameId`: Fortnite

`page`: 0 (0 - 100) \
`rank`: 1 (The rank is used if page and rank are given (e.g. 54 would still goto page 1 and start at rank 1) \
`showLiveSessions`: true
