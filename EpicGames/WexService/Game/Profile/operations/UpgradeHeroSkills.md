# UpgradeHeroSkills

**Description**: `Upgrades the skill level of a hero (+ pet in old versions)` \
**Profiles**: `profile0`

## Body

```js
{
    "heroItemId": "", // GUID of Hero to Upgrade
    "bIsInPit": false, // Whether the Hero is in the Pit
    "xpToSpend": 0 // Amount of Skill XP to spend on the Hero
}
```
