# UnlockRewardNode

**Description**: `unlocks a reward by its nodeId (used in winterfest)` \
**Profiles**: `athena`

## Body
```js
{
    "nodeId": "{nodeId}", // The rewards nodeId you want to open
    "rewardGraphId": "{guid}", // The RewardGraphs GUID
    "rewardCfg": "" // Can be left empty
}
```
